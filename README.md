# Mini Booking

Mini Booking made in Ember and Rails 5

## Setup
* `git clone https://github.com/akshay2408/mini_booking_sync.git`
* `cd mini_booking_sync`
* `bundle install`
* `rake db:create && rake db:migrate`

* `cd frontend`
* `npm install`

## Run Server
* `rails s`

## Run Tests
### Run Rails tests :
* `rspec`

### Run Ember tests :
* `cd frontend`
* `ember test`
* `To view test results in browser`
* `ember test --serve`
